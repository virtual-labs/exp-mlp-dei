Understanding the Multi-Layer Perceptron

Input Layer

Input features (sepal length, width, etc.) are fed into the network. These nodes do not perform calculation; they simply pass values to the next layer.

Hidden Layers

Neurons calculate a weighted sum of inputs plus a bias. This linear transformation allows the network to combine features in complex ways.

Activation (ReLU)

The Rectified Linear Unit applies a non-linear threshold: negative values become zero. This sparsity helps the network learn efficiently.

Forward Pass

Data flows from left to right. Each layer transforms the representation until it reaches the output. No learning happens here, just prediction.

Output Layer (Softmax)

The final layer converts raw scores (logits) into probabilities summing to 1. Each node represents a class confidence.

Prediction

The class with the highest probability is chosen as the model's final prediction. We compare this to the actual label to compute error.
